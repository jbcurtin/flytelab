Not Recommended
31.0 hrs last two weeks / 31.0 hrs on record (0.7 hrs at review time)
Posted: Feb 24 @ 8:09pm
Updated: Mar 3 @ 9:48pm

I want to like this game. I desperately do. I saw the reviews coming in and was super excited. Yet sadly this game just simply Does. Not. Run.

I have tried everything. Graphics Drivers Reinstalls, windows software reinstalls. Cache clearing. Steam launch options to launch in windowed or other modes. Esoteric fixes I found online and in reddit. I have done windows fire wall disables and third party antivirus disables.

This game Does. NOT. Run.

When I launch the game Anticheat runs just fine. The game initializes to a white screen.
And then nothing. It sits there for thirty seconds and is suspended by windows then crashes to desktop.

Every. Single. Time.

I've submitted a support request but if this isn't fixed relatively soon I am probably just going to refund the game and wait to hear if it ever is.

Update: I found a fix, unfortunately the fix was to totally wipe my PC and start fresh. So far it appears to be working, I will update this if my opinion changes but the fact this was required is infuriating.

MANTISxB Feb 28 @ 6:26pm 
So I have posted this on a few others comments and its helping my windows friends play. THIS WILL TAKE YOU OFFLINE as a heads up.

Go to the game folder > find "start_protected_game.exe" (should have the EAC bear logo on it.) and change the name to "start_protected_game.exe.backup"

Change "eldenring.exe" to "start_protected_game.exe"

launch the game.

The above makes EAC not load and the game plays without it. The game does a check to see if EAC is on, says it cant connect online, and just doesnt load it. This has given my windows friends MUCH better performance.

Im running this on linux and the only issue i have is a crash around 4-5 hours of continual play. Havent lost any progress as of yet however. Works perfectly online.

i7-7700K / GTX 1080 / 32GB RAM. perfect 60FPS. I had to do this trick to get it to play before the online was fixed and it worked like a champ.
