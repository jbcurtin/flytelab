My solution to White Screen Crash:
I updated my windows 10 to the latest version and then the game can launch properly. Such a weird bug, seems they used some windows API that was not in the old versions. There should be an os version requirement in the System Requirement section, not simply Windows 10, cuz old versions of windows 10 can't play the game.

Actual Review:
It's like Dark Souls 3 combined with Zelda Breath of the Wild, but mostly a Dark Souls game. Things are a bit confusing from start up, but getting familiar to veteran souls-like players after you beat Margit and enter the Stormveil castle. The whole castle is like a mini Dark Souls game, and all the level design goodies within it. But when out in the wild, it can be a totally different experience, you can find nomad camps, wandering optional bosses, some caves or dungeons to explore, it can be full of surprises just by wandering around.

If you are an old Dark Souls player, just buy it. If you are an average action game player, I suggest you watch some videos about souls-like combat to determine whether you'll like it or not.
Overall, I'm quite happy with the game, still get to explore more.
